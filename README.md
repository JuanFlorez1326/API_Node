# Available Scripts

## Install Packages

```bash
$ npm i
```
### or
```bash
$ npm install
```

## Generate the database and update
```
$ npx prisma migrate dev
```

## Running the app

```bash
# development
$ npm run start
```

## Create the app for production
```
$ npm run build
```
## Technologies
```
* TypeScript
* Node Js - Express Js
* PostgreSQL - ORM Prisma
* BCrypt - PassPort
* Json Web Token
```
